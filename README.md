# Workout-Tracker

 ![license](https://img.shields.io/badge/License-MIT-green.svg)

  ## Description

A fitness tracking app that allows a user to be able to view, create and track daily workouts. The application also allows you to track the name, type, weight, sets, reps, and duration of an exercise.
  
  This application was created using 
  - Node JS
  - Express
  - Mongo DB with Mongoose
